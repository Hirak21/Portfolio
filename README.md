# HRK // Future Systems Lab

Personal portfolio — a self-taught builder creating technology-driven systems for people, culture, and the future.

## Structure

```
portfolio/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── assets/
```

## Tech

- Plain HTML/CSS/JS — no build step, no framework
- Fonts: Bricolage Grotesque, Hanken Grotesk, IBM Plex Mono (Google Fonts)
- Theme: dark / light (persisted in localStorage)
- Canvas flow-field + CSS 3D wireframe core
- Responsive, reduced-motion safe
